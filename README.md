## 功能

- ✔️ Twitch、Youtube聊天室顯示。
- ✔️ Text-to-Speech聊天訊息朗讀。
- ✔️ 3種語言選擇，繁體中文、英文、日文。
- ✔️ 文字大小、行距、人名、表情符號大小可各自調較。
- ✔️ 透明聊天方塊可任意移動和調整大小。
- ✔️ UI在3秒後會自動隱藏，只剩下留言部份顯示，按一下留言左半部份，可恢復UI顯示。
